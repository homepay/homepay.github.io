Directory Service

HomePay’s Interoperable Directory Server is a scheme-agnostic routing and registry layer that sits between issuers and merchant/acquirer banks. Built to EMVCo principles, it accelerates new card scheme launches with message orchestration, directory data management, certified integrations, and production-ready operations.

Interoperability & Routing

Scheme-Agnostic Directory Layer

Orchestrates issuer ⇄ acquirer/merchant messaging with standards-aligned routing and policy controls.

  • Requestor ⇄ DS ⇄ ACS message handling
  • Directory metadata (BIN ranges, endpoints, keys)
  • Smart routing & rule-based policies
  • Versioning & backward compatibility
Scheme Enablement Toolkit
